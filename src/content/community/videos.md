### עיבוד שרת סטרימינג עם Suspense {/*streaming-server-rendering-with-suspense*/}

React 18 כולל גם שיפורים לביצוע רינדור בצד השרת באמצעות Suspense.

שרת זרימה יכול להשתמש HTML מקומפוננטות React בשרת, ולהזרים את ה-HTML עיבוד למשתמשים. ב-React 18 אפשר להשתמש ב-`Suspense` כדי לפרק את האפליקציה ליחידות עצמאיות קטנות יותר, אפשר להזרים בנפרד בלי לחסום את שאר האפליקציה. היא שהמשתמשים יראו את התוכן מוקדם יותר ויוכלו להתחיל אינטראקציה מהר יותר.

לצלילה עמוקה יותר, ראו את ההרצאה של [איש שונדאי](https://twitter.com/shaundai) כאן:

<YouTubeIframe src="https://www.youtube.com/embed/pj5N-Khihgc" title="YouTube video player" />

### קבוצת העבודה React הראשונה {/*the-first-react-working-group*/}

לקראת React 18 יצרנו את Working Group הראשון שלנו, כדי לשתף פעולה עם פאנל של מומחים, מפתחים, מתחזקי ספריות ואנשי חינוך. יחד בנינו את אסטרטגיית האימוץ ההדרגתי ושייפנו APIs חדשים כמו `useId`, `useSyncExternalStore` ו-`useInsertionEffect`.
