<Intro>

בגרסת ה-production המכווצת (ממוזער) של React אנחנו נמנעים משליחת הודעות שגיאה מלאות, כדי לצמצם את מספר הבתים שנשלחים ברשת.

</Intro>


אנחנו ממליצים מאוד להשתמש מקומית בגרסת הפיתוח כשמדבגים את האפליקציה, כי היא עוקבת אחרי מידע דיבוג נוסף ומספקת אזהרות מועילות על בעיות פוטנציאליות באפליקציה. עם זאת, אם מופיעה חריגה בזמן שימוש בגרסת ייצור, הודעת השגיאה תכלול רק קישור לתיעוד של שגיאה.
