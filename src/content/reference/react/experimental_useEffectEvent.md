- `react@experimental`

<Intro>

`useEffectEvent` הוא React Hook שמאפשר לחלץ לוגיקה לא-ריאקטיבית ל-[Effect Event.](/learn/separating-events-from-effects#declaring-an-effect-event)

```js
const onSomething = useEffectEvent(callback)
```

</Intro>
